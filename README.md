# Matrix to Complex Number Mapping

## Project Overview

This project implements a Residual Convolutional Neural Network (CNN) with attention mechanisms to map 8x8 binary matrices to complex number representations.

## Key Features

* Residual CNN architecture with attention blocks
* Custom complex distance loss function
* Training with learning rate scheduling
* Comprehensive visualization of results
* Mapping between matrices and integer vectors (X\_c)
